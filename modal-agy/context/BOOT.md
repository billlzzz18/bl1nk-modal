# BOOT.md — Sandbox Startup

## Checklist

1. Tools available: git, gh, agy, cargo, python3
2. Agent context: ~/.gemini/{AGENTS,SOUL,TOOLS,BOOT}.md
3. Plugins: installed via `agy plugin install`
4. Project: cloned to /tmp/

## First Steps

1. Check tools: `git --version && agy --help && cargo --version`
2. Install plugin if needed: `agy plugin install <url>`
3. Clone project: `git clone <url> /tmp/project`
