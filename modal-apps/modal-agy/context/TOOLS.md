## agy CLI Commands

| Command | Description |
|---------|-------------|
| agy plugin install <url> | Install plugin from GitHub |
| agy plugin list | List installed plugins |
| agy plugin validate <path> | Validate plugin structure |
| agy plugin explore | Explore available plugins |
| agy -p "prompt" | Run single prompt |
| agy | Interactive mode |
